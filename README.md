# FLIC Information Technology & Examination Analytics Dashboard

## Project Overview
This project focuses on building an interactive, data-driven analytics system (Dashboard) using **Streamlit** to support decision-making at the Foreign Language and Information Technology Center (FLIC). The solution processes, transforms, and visualizes complex data to track academic assessment performance and optimize operational and business efficiency.

*Note: For confidentiality and data security reasons, all charts and specific data figures have been omitted from the public repository, or simulated using generalized data.*

## Key Analytical Features
* **Examination Performance Analysis:** Visualizes pass/fail rates, grade distributions, and retake trends across various courses, modules, and testing periods to improve academic quality.
* **Business Operations & Revenue Tracking:** Evaluates financial performance by analyzing revenue breakdown across different training programs, IT services, and certification categories over time.
* **Interactive Dynamic Filters:** Empowers users to slice and dice data seamlessly by year, quarter, month, and specific exam types for real-time insights.

## Tech Stack & Skills Demonstrated
* **Programming Language:** Python (utilizing Pandas and NumPy for rigorous ETL processes and data wrangling).
* **Data Visualization:** Streamlit (coupled with Plotly / Matplotlib for building dynamic, interactive user interfaces).
* **Version Control:** Git & GitHub for professional project management.

## 📁 Repository Structure
* `app.py`: The main entry point launching the interactive Streamlit application.
* `pages/`: Contains separate multi-page files for dedicated examination and revenue dashboards.
* `scripts/`: Python scripts responsible for the underlying data processing logic and pipeline.
* `.gitignore`: Configured to exclude local data cache, logs, and sensitive source files.
